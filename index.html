Bem Vindo a Integração GitHub!

O Que é Github?

Github é uma plataforma de versionamento open source para usuários entusiastas ou não de programação e desenvolvimento de softwares diversos, a imaginação é o limite do que é possível fazer aqui!

Linguagens

Existem variadas linguagens e cada uma atende a uma necessidade diferente , isso depende muito do produto final. Algumas usam muito lógica matemática, outras são mais visuais e nem são consideradas linguagens e nesse grupo tem algumas bibliotecas confundidas também. é preciso estudar cada uma conforme a necessidade, aprender a síntaxe algo muito importante na programação, pois cada linguagem tem a sua. As tags, comentários igualmente se diferenciam mas com o tempo e muita dedicação podemos detectar só de olhar algumas linhas de código.

Depuradores

São eles que nos ajudam na comunicação entre máquina e desenvolvedor. é necessário uma equipe com bons profissionais desenvolvedores que apliquem bastante lógica com a ideia final do produto. após isso é incluido um monte de recursos legais como as extensões. com elas conseguimos fazer integrações como com o nosso Github!
